## Install on local
setup back before start front end
```
git clone https://github.com/w1216linw/blogweb.git
cd blogweb
yarn
```
add .env.local file with NEXT_PUBLIC_API_URL=http://localhost:4000/graphql inside
```
yarn dev
```
